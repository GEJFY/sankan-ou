## はじめに

**GRC Triple Crown（三冠王）** は、GRC（ガバナンス・リスク・コンプライアンス）分野の主要3資格を効率的に同時学習するための AI 搭載学習プラットフォームです。

### 対象資格

| 資格 | 正式名称 | テーマカラー |
|------|---------|------------|
| **CIA** | 公認内部監査人 (Certified Internal Auditor) | 赤 (#e94560) |
| **CISA** | 公認情報システム監査人 (Certified Information Systems Auditor) | シアン (#0891b2) |
| **CFE** | 公認不正検査士 (Certified Fraud Examiner) | 紫 (#7c3aed) |

### なぜ同時学習が有効なのか

CIA、CISA、CFEの3資格には **約40%の知識が重複** しています。リスク管理、内部統制、ガバナンス、コンプライアンスといった共通テーマを横断的に学ぶことで、以下のメリットがあります。

- **学習時間の大幅短縮**: 重複する知識を一度の学習で複数資格に活用
- **理解の深化**: 異なる視点から同じ概念を学ぶことで、より深く理解
- **合格率の向上**: シナジー効果により、各資格の合格確率が相互に高まる

### 主な機能

- **FSRS v6 アルゴリズム** による科学的な間隔反復学習
- **AI Tutor** による6段階レベル別の概念解説、3資格比較、ソクラテス式対話
- **AI 自動生成問題** によるトピック別の実力確認
- **本番形式の模擬試験** で各資格のフォーマットに対応
- **シナジー学習** で資格間の共通知識を効率的に統合
- **受験戦略パネル** で合格確率予測と最適な学習計画を提案
- **ゲーミフィケーション** でモチベーションを維持

## ダッシュボード

ログイン後、最初に表示されるメイン画面です。登録済みの全資格の学習状況を一目で把握できます。

### 初回利用時（オンボーディングガイド）

コースに未登録の場合、4つのステップガイドが表示されます。

| Step | 内容 | 説明 |
|------|------|------|
| 1 | 設定 | 学習する資格を登録 |
| 2 | SRS学習 | フラッシュカードで基礎知識を記憶 |
| 3 | 問題演習 | トピック別の問題で実力チェック |
| 4 | 模擬試験 | 本番形式で合格力を測定 |

### 表示される情報

- **コース別プログレスリング**: 各資格ごとのカード習得率を円形グラフで表示
  - 全カード数、習得済み数、本日の復習予定数を確認可能
- **ストリークバッジ**: 連続学習日数の表示
- **今日のタスク**: 本日復習予定のカード数と、すでに学習済みのカード数
- **弱点トピック**: 習得率が低いトピックの一覧（優先的に学習すべき分野）
- **合格確率**: 各資格の現在の合格確率をシグモイド関数で算出
- **シナジーマップ**: 資格間の共通テーマの重複状況を可視化
- **学習履歴グラフ**: 直近の学習推移を表示
- **最近の模擬試験**: 過去に受験した模擬試験のスコアと合否結果

## コース登録と設定

### コースの登録

1. サイドバーから「**設定**」ページに移動
2. 「**利用可能なコース**」セクションに未登録のコース（CIA / CISA / CFE）が表示される
3. 登録したいコースの「**登録**」ボタンをクリック
4. 登録完了後、システムが自動的に学習カードを割り当てます

### 目標記憶率の調整

コースごとに「**目標記憶率（desired retention）**」を設定できます。これは復習時に正しく思い出せる確率の目標値です。

| 設定値 | 特徴 | 推奨用途 |
|--------|------|---------|
| **80%** | 復習頻度が低い | 忙しい方、長期計画向け |
| **85%** | バランス型 | 一般的な学習ペース |
| **90%** (推奨) | 確実な定着を目指す | 標準的な受験準備 |
| **95%** | 高頻度の復習 | 試験直前の追い込み |

目標記憶率を高くするほど復習の間隔が短くなり、復習頻度が上がります。逆に低くすると復習の間隔が広がり、1日あたりの復習量が減ります。

### コースの登録解除

登録コースの右側にあるゴミ箱アイコンをクリックすると、そのコースの登録を解除できます。

## SRS学習（間隔反復学習）

SRS（Spaced Repetition System）は、科学的に証明された記憶定着手法です。本アプリでは **FSRS v6（Free Spaced Repetition Scheduler）** アルゴリズムを採用しています。

### 学習の流れ

1. サイドバーから「**SRS学習**」を選択
2. 復習予定のカードが自動的に表示される
3. **カードの表面**（質問）を読む
4. `Space` キーまたはカードをクリックして **裏面**（解答）を表示
5. 自分の思い出し具合を **4段階** で評価

### 評価の4段階

| 評価 | キー | 意味 | 次の復習タイミング |
|------|-----|------|------------------|
| **Again** | `1` | 全く思い出せなかった | すぐに再表示 |
| **Hard** | `2` | 思い出せたが自信がない | 短い間隔で復習 |
| **Good** | `3` | 正しく思い出せた | 通常の間隔で復習 |
| **Easy** | `4` | 即座に思い出せた | 長い間隔で復習 |

### セッション完了

全てのカードの復習が終わると、セッション完了画面が表示されます。

- 復習したカード枚数
- 正答率（Good以上の割合）

「**次のセッションを開始**」ボタンで、新たに割り当てられたカードを続けて学習できます。

### FSRS アルゴリズムについて

FSRSは各カードの記憶状態を個別にモデル化し、最適な復習タイミングを計算します。評価の結果に基づいて、以下のパラメータが動的に調整されます。

- **安定性 (Stability)**: 記憶がどの程度定着しているか
- **難易度 (Difficulty)**: そのカードの難しさ
- **復習間隔 (Interval)**: 次に復習するまでの日数

## AI Tutor

GRC 分野に特化したAI講師と対話形式で学習できます。

### 5つのモード

| モード | 説明 | 用途 |
|--------|------|------|
| **Q&A** | 自由に質問できるチャット | GRC分野の疑問に何でも回答 |
| **解説** | 6段階レベル別の概念解説 | 初心者から専門家まで、理解度に合わせた解説 |
| **比較** | CIA/CISA/CFE 3資格比較表 | 同じ概念が各資格でどう扱われるか整理 |
| **ソクラテス式** | 対話形式で理解を深化 | 問答形式で考えを深め、理解を確認 |
| **ブリッジ** | 資格間の知識橋渡し | ある資格の知識を別の資格に応用する方法 |

### 解説レベル（6段階）

| レベル | 対象 | 特徴 |
|--------|------|------|
| 1 | 小学生（10歳） | 身近な例え話、専門用語なし |
| 2 | 中学生（15歳） | 基本概念から丁寧に、簡単な専門用語は使用可 |
| 3 | 高校生（18歳） | 論理的に構造化、因果関係や背景知識も含む |
| 4 | 大学生・社会人 | 専門用語を使い、理論と実務の両面から解説 |
| 5 | 実務経験者 | 実務での適用場面やケーススタディを含む |
| 6 | 公認会計士・専門家 | 規準の条文・判例・国際基準との比較 |

### 機能詳細

- **Markdown 対応**: 表、箇条書き、太字、見出しなどを含むリッチなフォーマットで回答
- **SSE ストリーミング**: 回答がリアルタイムで逐次表示
- **音声読み上げ (TTS)**: アシスタントの回答を日本語で読み上げ（各メッセージのスピーカーアイコンをクリック）
- **フォローアップ質問**: 各回答の末尾に関連する質問が3つ提案され、クリックするだけで追加質問が可能
- **会話履歴のクリア**: ゴミ箱アイコンで会話をリセット

## 問題演習

トピック別の四肢択一問題で実力を確認します。データベースに保存済みの問題を優先的に出題し、不足分はAIがリアルタイムで生成します。

### 使い方

1. サイドバーから「**問題演習**」を選択
2. **コースを選択**: CIA / CISA / CFE から選ぶ
3. **トピックを選択**: 選択したコースに含まれるトピック一覧から出題範囲を指定
4. **問題数を選択**: 5問 / 10問 / 15問 / 20問 / 30問 / 50問
5. **出題ソースを選択**:
   - **自動（DB優先）**: 保存済み問題を優先し、不足分をAI生成
   - **AI生成のみ**: 全問を新規に AI が生成
6. 「**開始**」ボタンをクリック

### 回答の流れ

1. 問題文と4つの選択肢が表示される
2. 選択肢をクリックして回答を選ぶ
3. 「**回答する**」ボタンで確定
4. **正解・不正解の表示**: 正解は緑、不正解は赤でハイライト
5. **各選択肢の解説**: なぜその選択肢が正しい/間違いかの説明
6. **問題全体の解説**: より詳しい背景知識の解説
7. 「**次の問題**」で次に進む

### 演習完了

全問回答後、正答数と正答率が表示されます。「**もう一度**」ボタンでトピック選択画面に戻ります。

## 模擬試験

各資格の本番試験フォーマットに準拠した模擬試験を受験できます。

### 各資格の試験仕様

| 資格 | 本番問題数 | 制限時間 | 合格基準 | 構成 |
|------|----------|---------|---------|------|
| **CIA** | 3パート 各125問 | 各パート別 | 75% | Part 1-3 |
| **CISA** | 150問 | 240分 | 70% | Domain 1-5 |
| **CFE** | パート別 | パート別 | 60% | Section 1-4 |

### 使い方

1. サイドバーから「**模擬試験**」を選択
2. **資格を選択**: CIA / CISA / CFE
3. **科目を選択**: 全科目（ランダム出題）または特定のパート/ドメイン/セクション
4. **練習問題数を選択**: 5 / 10 / 20 / 30 / 50 / 100 問
5. 「**模擬試験を開始**」ボタンをクリック

### 試験中の画面

- **上部バー（スティッキー）**:
  - 選択した資格バッジ
  - 現在の問題番号（例: 3 / 20）
  - 残り時間のカウントダウン
  - 「**提出**」ボタン
- **問題番号インジケーター**: 全問の回答状況を色分けで表示
  - 青: 現在の問題
  - 緑: 回答済み
  - グレー: 未回答
- **問題エリア**: 問題文と4つの選択肢
- **ナビゲーション**: 前の問題 / 次の問題 ボタンで自由に問題間を移動可能

### 結果画面

試験終了後（手動提出または時間切れ）に以下が表示されます。

- **スコア（パーセント）**: 大きく表示、合格なら緑、不合格なら赤
- **正解数 / 全問数**
- **合否判定**: 合格基準との比較
- **問題別レビュー**: 全問の正誤と解説を確認可能
- 結果はデータベースに自動保存され、ダッシュボードの「最近の模擬試験」に表示されます

## シナジー学習

CIA / CISA / CFE に共通するテーマのカードを横断的に学習します。一度の学習で複数資格の知識を同時に強化できます。

### シナジーエリア

画面上部に、資格間で重複するテーマ領域がタグとして表示されます。例:

- リスク管理
- 内部統制
- ガバナンス
- コンプライアンス
- 不正リスク対応

各タグには重複率（%）と関連する資格がカラードットで表示されます。

### 使い方

1. サイドバーから「**シナジー学習**」を選択
2. **対象資格を選択**: CIA / CISA / CFE から1つ以上を選択（トグル方式）
3. **カード枚数を選択**: 5 / 10 / 15 / 20 / 30 / 50 枚
4. 「**シナジー学習を開始**」ボタンをクリック

### 学習中の画面

- **プログレスバー**: 全資格の色のグラデーション（赤→シアン→紫）で進捗表示
- **コースバッジ**: 現在のカードがどの資格に属するかを表示
- **SYNERGY バッジ**: 複数資格に関連するカードに付与
- **タグ**: そのカードに関連するキーワード
- SRS学習と同様に、フリップして4段階で評価

### キーボード操作

| キー | 動作 |
|------|------|
| `Space` / `Enter` | カードをフリップ |
| `1` | Again（全く思い出せなかった） |
| `2` | Hard（かなり悩んだ） |
| `3` | Good（少し考えて思い出せた） |
| `4` | Easy（即座に思い出せた） |

## メディア学習（スライド / 音声）

AI がトピックに基づいた学習コンテンツを自動生成します。

### スライド生成

1. サイドバーから「**スライド/音声**」を選択
2. モードタブで「**スライド生成**」を選択
3. **コースを選択**: CIA / CISA / CFE
4. **カテゴリ（科目）を選択**: ドロップダウンから出題分野を選ぶ
5. **スライド枚数を選択**: 5 / 8 / 10 / 15 枚
6. **追加プロンプト（任意）**: 内容を詳細にカスタマイズしたい場合に入力
   - 例: 「5つの構成要素を図解で詳しく」
7. 「**生成**」ボタンをクリック（生成に約10-20秒）

### スライドの閲覧

- 前のスライド / 次のスライド ボタンで移動
- ドットインジケーターで任意のスライドに直接ジャンプ可能
- 各スライドにはノート（補足説明）が付く場合があります
- **ダウンロード**: 「ダウンロード (JSON)」ボタンで生成したスライドを JSON 形式で保存

### 音声スクリプト生成

1. モードタブで「**音声スクリプト**」を選択
2. コースとカテゴリを選択
3. 「**生成**」ボタンをクリック
4. AI が学習用の音声台本をセクション分けして自動作成

各セクションには以下が含まれます:
- **セクションタイトル**
- **スクリプト本文**: 読み上げ用のテキスト
- **理解度チェック**: そのセクションの内容を確認するための質問

> **注意**: TTS 音声再生機能は将来バージョンで実装予定です。現在はスクリプトのテキスト表示のみです。

## 受験戦略パネル

学習データを分析し、各資格の合格確率・予測スコアを算出。最も効率的な受験順序と学習時間配分を提案します。

### 表示される情報

#### 推奨受験順序

合格確率が高い資格から順に受験する戦略を提案します。先に合格した資格の知識がシナジー効果として次の資格の学習に活かされるため、この順序が最も効率的です。

#### サマリー統計

| 指標 | 説明 |
|------|------|
| **推定残り学習時間** | 未習得カードを全て学習するのに必要な推定時間（時間単位） |
| **習得済みカード** | FSRS 評価で Good 以上を一定回数以上獲得したカード数 |
| **残りカード** | まだ習得に至っていないカード数 |

#### 資格別戦略カード

各資格について以下の詳細情報を表示:

- **合格確率**: 現在の学習データから算出した本試験の合格確率
- **予測スコア**: SRS カードの習得率と正答率から予測した試験スコア
- **残り学習時間**: 未習得カードの学習に必要な推定時間
- **進捗**: 全カードのうち習得済みカードの割合（プログレスバー付き）
- **優先改善トピック**: 習得率が低いトピックを優先度順に表示（最大3件）
- **推奨コメント**: AI が生成する学習アドバイス

## ゲーミフィケーション（実績）

学習のモチベーションを維持するためのXP（経験値）、レベル、バッジ、デイリーミッションシステムです。

### XP（経験値）の獲得

学習活動に応じてXPを獲得できます。

| 活動 | 獲得 XP |
|------|---------|
| カードレビュー (Again) | 5 XP |
| カードレビュー (Hard) | 10 XP |
| カードレビュー (Good) | 15 XP |
| カードレビュー (Easy) | 25 XP |
| 問題正解 | 20 XP |
| シナジーボーナス | +10 XP |

### レベルシステム

累積 XP に応じて最大30段階のレベルが上がります。レベルが上がるほど次のレベルに必要な XP が増加します。

- **Lv1**: 0 XP
- **Lv5**: 1,000 XP
- **Lv10**: 5,200 XP
- **Lv20**: 36,000 XP
- **Lv30**: 175,000 XP

各資格ごとの XP 内訳もバーグラフで確認できます。

## FAQ（よくある質問）

### Q: 新規登録後、カードが表示されません

**A**: コースへの登録が必要です。サイドバーの「**設定**」に移動し、学習する資格（CIA / CISA / CFE）を登録してください。登録後、システムが自動的に未学習カードを割り当てます。

### Q: AI機能（AI Tutor、問題生成、スライド生成）が使えません

**A**: AI機能を利用するには、バックエンド側で Azure AI Foundry または Gemini の API キーが正しく設定されている必要があります。環境変数 `AZURE_AI_ENDPOINT`、`AZURE_AI_KEY` などが `.env` ファイルに設定されていることを確認してください。

### Q: テーマ（ダーク/ライトモード）の変更が反映されません

**A**: テーマ設定はブラウザの `localStorage` に保存されます。反映されない場合は、ブラウザをリロード（F5 / Ctrl+R）してください。シークレットモードでは設定が保持されませんのでご注意ください。

### Q: 模擬試験の結果が保存されていないようです

**A**: 模擬試験の結果は試験終了時（手動提出または時間切れ）に自動保存されます。ブラウザを閉じたり、ページを離れた場合は保存されません。試験中は必ず「**提出**」ボタンで結果を確定してください。

### Q: SRS学習で「セッション完了」と表示されるが、カードがすべて復習済みなのか？

**A**: はい、その日に復習が予定されていたカード（due cards）はすべて復習済みです。新規の未学習カードは翌日以降に追加されるか、「次のセッションを開始」をクリックすると新たなカードが割り当てられる場合があります。

### Q: 問題演習で「問題生成に失敗しました」と表示されます

**A**: 以下を確認してください:
1. API キーが正しく設定されているか
2. 選択したコースにトピック（シードデータ）が存在するか
3. ネットワーク接続が安定しているか

大量の問題（30問以上）を生成する場合は、バッチ処理のため30-60秒程度かかることがあります。

### Q: 複数デバイスで学習データを同期できますか？

**A**: はい。学習データはすべてサーバー側（PostgreSQL データベース）に保存されるため、同じアカウントでログインすれば、どのデバイスからでも同じ学習進捗にアクセスできます。ただし、テーマ設定（ダーク/ライト）はデバイスごとのローカル設定です。
